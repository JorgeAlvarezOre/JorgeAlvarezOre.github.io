Mi portafolio

Clasificador de imágenes (Proyecto de TensorFlow)

Modelo de Red Neural Convolucional (CNN) para clasificar imágenes de especies botánicas comunes en Río Blanco - Lima - Perú. Modelo entrenado a partir de un conjunto de datos de búsqueda propio de 3619 fotografías de imágenes de Google divididas en 17 clases botánicas. La aplicación de este código agilizará la clasificación preliminar en campo, especialmente en estudiantes donde el reconocimiento de especies botánicas es aún limitado. El modelo utiliza la API de Keras, la plataforma Tensorflow y las bibliotecas NunPy y Matplotlib.

Pizza reporte de ventas (Proyecto de SQL + Power BI)

Consultas SQL y dashboard de Power BI para analizar los indicadores clave de rendimiento (KPI) de una pizería ficticia y mostrar algunos gráficos para obtener información sobre el rendimiento empresarial y comprender las tendencias del negocio.
